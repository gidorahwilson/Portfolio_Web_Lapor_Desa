# Portfolio_Web_Lapor_Desa

# 🏡 Lapor Desa App

A simple web application for village complaint reporting system built with **Sinatra (Ruby)** and **MySQL**.

---

## 🚀 Features

- User Login System
- Submit complaint reports
- MySQL database integration
- Simple session-based authentication
- Environment variable support (.env)

---

## 🧱 Tech Stack

- Ruby (Sinatra Framework)
- MySQL
- mysql2 gem
- dotenv gem
